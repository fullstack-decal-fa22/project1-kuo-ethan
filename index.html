Graph Algorithms Visualizer

After studying graph algorithms in CS61B, I sought to apply what I had learned by creating a graph algorithms visualizer. The visualizer is interactive and enables users to create their own graphs and run various pathfinding and spanning algorithms on them. The visualizer features Dijkstra's, A* (with Euclidean and Manhattan heuristics), Prim's, and Kruskal's algorithm.
